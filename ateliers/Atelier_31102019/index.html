Ordre du jour : Fonctions, types de variables, boucles et conditions

F.A.Q

Question 1 :
Johnny : A quel moment utiliser getElement... ou querySelector ?

querySelector est plus récent que getElement.
querySelector est totalement flexible.
- getElementById retourne TOUJOURS une seule valeur, l'élément HTML avec l'id renseigné.
- getElementsByClassName retourne TOUJOURS un tableau, composé des éléments HTML qui possède la classe en question.
- querySelector retourne UN SEUL élément, le 1er qui correspond à la requête.
Exemple : querySelector("#section1 .container h2") retourne le 1er h2 qui est dans un container qui est lui-même dans la section 1.
- querySelectorAll retourne TOUS les éléments qui correspondent à la requête dans un tableau (même s'il n'y a qu'un seul élément !)



Question 2 :
Arnaud : Comment marche "console.log" ?

console.log écrit dans la console du navigateur. Si entre une fonction dans le .log, il écrira le résultat de la fonction, si on entre du texte, il écrira le texte, etc. Attention : le code s'exécute même si on fait console.log(code);



Question 3 :
Johnny : Quand utiliser var, let et const?

var existait avant let et const, let et const sont plus précis. Const permet déclarer une "variable" constante. Let est une variable dont la valeur peut être ré-assignée



Question 4 :
Arnaud : Qu'est-ce que parseInt ?

La fonction parseInt() analyse une chaîne de caractère fournie en argument et renvoie un entier exprimé dans une base donnée.

Types de variables

Il existe plusieurs types de variable. Par exemple ; number, string, boolean, array, objets. 
          let texte = "";
          les guillemets permettent de créer un string
      
          let number = 50;
          Si on écrit juste un entier, la variable sera de type number.
      
          let boolean = true;
          Si on écrit juste true ou false, ça crée un boolean.
      
          let array = [];
          Les crochets permettent de déclarer un tableau.
      
          let objet = {};
          Les accolades permettent de déclarer un objet.
Les différents types ne peuvent pas être comparés correctement dans les if.

Fonctions

une fonction doit d'abord être définie, puis ensuite appelée.
On définit la fonction avec le mot-clé "function".
On lui donne ensuite un nom (exemple: function maFonction)
Puis éventuellement des paramètres, et enfin, on met le code à exécuter entre accolades.
